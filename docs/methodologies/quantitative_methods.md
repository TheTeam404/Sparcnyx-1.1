# Quantitative Analysis Methodologies in SpectraMapper Pro

Beyond identifying elements, LIBS can be used for quantitative analysis, such as determining elemental concentrations or estimating plasma parameters. SpectraMapper Pro includes tools to facilitate some of these quantitative aspects.

## 1. Plasma Temperature Estimation via Boltzmann Plot

Under the assumption of Local Thermodynamic Equilibrium (LTE) in the LIBS plasma, the relative intensities of emission lines from the same element and ionization stage can be used to determine the plasma excitation temperature (T<sub>e</sub>).

*   **Principle:** The intensity (I<sub>ki</sub>) of an emission line resulting from a transition from an upper energy level (E<sub>k</sub>) to a lower energy level (E<sub>i</sub>) is given by:

    I<sub>ki</sub> = F * C * (A<sub>ki</sub> * g<sub>k</sub> / λ<sub>ki</sub>) * exp(-E<sub>k</sub> / (k<sub>B</sub> * T<sub>e</sub>))

    Where:
    *   `F`: An instrumental factor (assumed constant for lines close in wavelength).
    *   `C`: Concentration of the emitting species.
    *   `A_ki`: Transition probability for the k → i transition.
    *   `g_k`: Statistical weight (degeneracy) of the upper energy level E<sub>k</sub>.
    *   `λ_ki`: Wavelength of the emitted photon.
    *   `E_k`: Energy of the upper level.
    *   `k_B`: Boltzmann constant.
    *   `T_e`: Plasma excitation temperature.

    This equation can be linearized by taking the natural logarithm:

    ln(I<sub>ki</sub> * λ<sub>ki</sub> / (A<sub>ki</sub> * g<sub>k</sub>)) = -E<sub>k</sub> / (k<sub>B</sub> * T<sub>e</sub>) + ln(F * C)

    This is in the form of `y = m*x + c`, where:
    *   `y = ln(I_ki * λ_ki / (A_ki * g_k))`
    *   `x = E_k`
    *   `m = -1 / (k_B * T_e)` (slope)
    *   `c = ln(F * C)` (intercept)

*   **Procedure in SpectraMapper Pro:**
    1.  **Line Selection:** The user must select multiple (typically 3 or more, ideally 5+) emission lines from the *same element and the same ionization stage*. These lines should have accurately known atomic parameters: `A_ki`, `g_k`, and `E_k` (obtained from the elemental identification module and its database). The selected lines should also span a reasonable range of `E_k` values for a more reliable fit.
    2.  **Intensity Measurement:** The intensities `I_ki` (typically peak height or fitted peak area after baseline correction) and wavelengths `λ_ki` are taken from the experimental spectrum.
    3.  **Calculation:** For each selected line, the software calculates the `y` and `x` values as defined above.
    4.  **Plotting:** A "Boltzmann Plot" is generated by plotting `y` versus `x`.
    5.  **Linear Fit:** A linear regression (least squares fit) is performed on these plotted points.
    6.  **Temperature Derivation:** The plasma temperature `T_e` is calculated from the slope `m` of the fitted line:
        `T_e = -1 / (m * k_B)`
        (Units must be consistent, e.g., if E<sub>k</sub> is in cm<sup>-1</sup>, k<sub>B</sub> needs to be in cm<sup>-1</sup>/K).
    7.  **Results:** The calculated T<sub>e</sub> and the R² value (goodness-of-fit) of the linear regression are displayed. A high R² value (e.g., > 0.95) suggests a good linear relationship and more reliable temperature estimation.

*   **Assumptions and Limitations:**
    *   **LTE:** The validity of the Boltzmann plot method relies heavily on the assumption of LTE.
    *   **Optically Thin Plasma:** Assumes no self-absorption for the selected lines. Strong resonance lines are often self-absorbed and should be avoided or corrected.
    *   **Accurate Atomic Data:** The accuracy of T<sub>e</sub> is directly dependent on the accuracy of the A<sub>ki</sub>, g<sub>k</sub>, and E<sub>k</sub> values from the database.
    *   **Intensity Measurement Accuracy:** Precise baseline correction and intensity measurement are critical.

## 2. Saha-Boltzmann Plot (Optional/Future)

The Saha-Boltzmann plot is an extension that can be used to determine both electron temperature and electron density (N<sub>e</sub>) if lines from two different ionization stages of the same element are used. This is a more complex method and may be considered for future implementation.

## 3. Calibration Curves for Concentration Analysis (Future)

This is a common quantitative method in LIBS where the intensity of a specific elemental line is correlated with the known concentration of that element in a set of standard reference materials.

*   **Principle:**
    1.  Prepare or obtain a set of standard samples with known and varying concentrations of the analyte(s).
    2.  Acquire LIBS spectra from each standard under identical experimental conditions.
    3.  Select an appropriate analysis line for the analyte (strong, interference-free, not prone to self-absorption at the concentrations of interest).
    4.  Measure the intensity (or area) of this line for each standard, after appropriate preprocessing (e.g., baseline correction, normalization to an internal standard or total spectral intensity).
    5.  Construct a calibration curve by plotting Line Intensity vs. Concentration.
    6.  Fit a model (e.g., linear, polynomial) to this curve.
    7.  For an unknown sample, acquire its LIBS spectrum under the same conditions, measure the intensity of the analysis line, and use the calibration curve to determine its concentration.

*   **Implementation in SpectraMapper Pro (Conceptual):**
    *   Allow users to input concentration data for a set of loaded spectra (standards).
    *   Provide tools to select an analyte and its analysis line.
    *   Plot intensity vs. concentration and perform regression.
    *   Allow users to predict concentrations for unknown samples based on the built curve.

Quantitative analysis in LIBS can be complex, often requiring careful experimental design, method validation, and consideration of matrix effects. SpectraMapper Pro aims to provide tools that assist in these established methodologies.
